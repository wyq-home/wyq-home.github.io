# How to take notes

This is the guide on how to take notes with pen and annotate pdf file.

# Taking notes with pen

  • Green color: module partitioning (e.g. title, sub-title, modules' title)
  • Black color: general contents
  • Red/Blue color: work with black-colored notes to make notes more readable and make important contents more obvious

# Annotating pdf

  • Green highlight: module partitioning
  • Yellow highlight: general contents
  • Grey highlight: not read yet
  • Red pen / Red arrow: indicate interal relationship
