# Notes

首先是编程类的必修：

  • 编程基础 (15-112: Fundamentals of Programming)
  • 指令式编程 (15-122: Principles of Imperative Computation)
  • 函数式编程 (15-150: Principles of Functional Programming)
  • 数据结构 (15-210: Parallel and Sequential Data Structures and Algorithms)
  • 系统 (15-213: Introduction to Computer Systems)
  • 算法 (15-451: Algorithm Design and Analysis)

然后是数学类的必修：

  • 数学基础 (15-151: Mathematical Foundations for Computer Science)
  • 离散数学 (15-251: Great Ideas in Theoretical Computer Science)
  • 微积分 (21-120: Differential and Integral Calculus)
  • 21-122: Integration and Approximation
  • 高等代数类，2 选 1:
    • 21-241: Matrices and Linear Transformations
    • 21-242: Matrix Theory
  • 概率类，4 选 1:
    • 15-259: Probability and Computing
    • 21-325: Probability
    • 36-218: Probability Theory for Computer Scientists
    • 36-225: Introduction to Probability Theory and 36-226: Introduction to Statistical Inference

以下都是 n 选 1。

然后是编程和逻辑类，n 选 1:

学习一门这类课程的目的是学会：Rigorous, formal reasoning about programs or systems, rooted in their logical foundations.

  • 编程语言 (15-312: Foundations of Programming Languages)
  • 15-314: Programming Language Semantics
  • 15-316: Software Foundations of Security and Privacy
  • 15-317: Constructive Logic
  • 15-414: Bug Catching: Automated Program Verification and Testing
  • 15-424: Foundations of Cyber-Physical Systems
  • 程序分析 (17-355: Program Analysis)
  • 80-413: Category Theory

系统类，n 选 1:

  • 操作系统 (15-410: Operating System Design and Implementation)
  • 编译远离 (15-411: Compiler Design)
  • 并行 (15-418: Parallel Computer Architecture and Programming)
  • 分布式系统 (15-440: Distributed Systems)
  • 网络 (15-441: Computer Networks)
  • 数据库 (15-445: Database Systems)

AI 类，n 选 1：

  • 机器学习 (10-315: Introduction to Machine Learning (10-601 is no longer allowed))
  • NLP (11-411: Natural Language Processing)
  • AI (15-381: Artificial Intelligence: Representation and Problem Solving)
  • 神经计算 (15-386: Neural Computation)
  • 16-384: Robot Kinematics and Dynamics)
  • 视觉 (16-385: Computer Vision)

找一个自己的领域，n 选 1：

  • 计算机生物 (02-251: Great Ideas in Computational Biology)
  • 05-391: Designing Human-Centered Software
  • 计算机音乐 (15-322: Introduction to Computer Music)
  • 安全 (15-330: Introduction to Computer Security)
  • 计算理论 (15-455: Undergraduate Computational Complexity)
  • 图形学 (15-462: Computer Graphics)
  • 软件工程 (17-313: Foundations of Software Engineering)

自主方向：

【CS 任选】其他 CS 课程，任选 2 门：

  • Computer Science (15-xxx)
  • Computational Biology (02-xxx)
  • Human-Computer Interaction (05-xxx)
  • Machine Learning (10-xxx)
  • Language Technologies (11-xxx)
  • Robotics (16-xxx)
  • Software Research (17-xxx)

【Concentration/Minor】要不选一个 CS 里的 Concentration，要不选一个其他学科的 Minor

CS Concentrations 包括：

  • Algorithms & Complexity
  • Computational Biology
  • Computer Systems
  • Human-Computer Interaction
  • Machine Learning
  • Programming Languages
  • Security and Privacy
  • Software Engineering

通识教育：

Technical Communications，n 选 1。

其他科学和工程学科的导论，任选 4 门。

人文艺术类，任选 63 unit (约 7 门)。

Free Electives：再随便选一门课，限：Physical Education, Military Science...。
