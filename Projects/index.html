# Howto

Before turning the status of a project from "on-going" to "finished":

  1. Make sure to add all stuffs about the project to the project, especially writeup.
  2. In README of the project, talk about some key points about the project and how to setup the project.
    • If the setup process need some large files which might disappear in the future, talk about it in README and copy that file to my storage space (with a unique name).
  3. Check the setup process from scratch.
  4. Add the link of the project to this repository.
  5. Remove local copy, since GitHub is enough.
